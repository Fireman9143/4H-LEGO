The Pixy Cam 2 camera module

The 2025 4H Illinois State Competition had AI as a main theme. The Pixy Cam 2 is a camera with its own program that allows the EV3 to respond to vector images, shapes, colors, and barcodes. At the time of making this site, RobotShop has the Pixy 2.1 for LEGO, which includes the cable with LEGO's proprietary RJ12 connector Adafruit female socket. While we're on the subject, as we expand into more capable robots you may need more custom wire connections. Mindsensors (male plug) has many third party parts that are compatible with EV3.

Pixy2 has software that needs to be installed on the computer, separate from VS Code. The software will be used to configure the camera. Be sure of which version of software you are downloading. The Pixy2 documentation is good, though it's not easy to follow. Here are some guidelines:

    Pixy2 for Labview or Scratch:
  1. Go to Pixy2 Downloads
  2. Select and download the PixyMon V2 program and the Pixy2 LEGO firmware
  3. Download the LEGO Blocks version 2.0.9 file (needed to make the LabView and Scratch blocks)
  4. Follow the uploading firmware instructions to install the LEGO firmware on the camera
  5. Open LabView and follow the LEGO Mindstorms EV3 Home Edition instructions for installing the block
    Pixy2 for Python:
  1. Go to Pixy2 Downloads
  2. Select and download the PixyMon V2 program and the Pixy2 general firmware
  3. Go to the Python tutorial page and scroll down to Installing pixycamev3: 2. Adding the API to your project folder, and look for the github link to find the pixy2.py file to copy into your project folder as described.
  4. Follow the uploading firmware instructions to install the general firmware on the camera
